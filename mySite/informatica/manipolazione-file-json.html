Manipolazione File Json con Python

Indice

Cos'e' un file JSON?

I file Json vengono utilizzati per lo scambio di dati di qualsiasi tipo. Sono file contententi dei tag e ad ognuno di questi viene associato un dato o una serie di dati. Un esempio potrebbe essere: {"nome": 'Giuseppe'; "cognome": 'Cosenza'} e cosi' via. Da qui i dati sono facilmente estraibili e soprattutto ordinati. Il campo di utilizzo e' molto vasto, per esempio pensate ai dati di questa attuale pandemia. Un altro esempio e' il salvataggio di dati di gioco, quindi parliamo del vostro punteggio, del livello del vostro personaggio e cosi' via. Viene quindi "compilato" un unico file Json con tutti i tag e i dati per poi essere caricato sulla rete ed essere utilizzato a vostro piacimento. Ovviamente potete creare anche il vostro file senza caricarlo in rete.

Come manipolare un file Json?

Facilissimo, per prima cosa dovete sapere dove si trova il file Json che vi interessa. Quindi se state lavorando con file in locale, ovvero sul vostro pc, dovrete specificare la posizione del file in questione. Se state lavorando con file online e quindi caricati sulla rete, il piu' delle volte e' possibile prelevare dal sito stesso il collegamento del file. In breve il sito vi fornira' un link da incorporare nel vostro codice e che permettera' il collegamento col vostro file (e' possibile anche scaricare il file in questione e quindi utilizzarlo come file locale). Bene, ho dato per scontato che se state leggendo questo "articolo" e' perche' avete delle basi di programmazione o conoscete almeno un minimo di cosa stiamo parlando. Per quanto riguarda il linguaggio di programmazione, non dovrebbero esserci limitazioni su cio' che utilizzate. In questo caso io utilizzo python ma dovrebbe essere la stessa cosa anche per gli altri linguaggi. Per prima cosa dovete importare la libreria json in questo modo >> import json . Successivamente dovete caricare il file, quindi vi basta instanziare una variabile (su python non bisogna specificare il tipo e dovrebbe riconoscerla come dizionario) e assegnarli >> vostra-variabile = json.load(open("nome-del-file.json")) . In questo modo potete lavorare col vostro file. Un modo veloce per poterlo fare e' il seguente >> vostra-variabile["tag-dati-json"] . Se per esempio volete prelevare piu' dati dello stesso tag, potete inserire quel codice in un ciclo, farlo andare fino alla lunghezza del file e quindi assegnarlo ad un array o a cio' che piu' vi serve.
